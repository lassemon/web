# web livereload demos

#### pre-install
- install node
 - https://nodejs.org/
- install yarn *(or use npm)*
 - https://yarnpkg.com/en/
- install global gulp
 - `$ yarn global add gulp@latest`

### Installation
`$ yarn install`

### builing app
`$ gulp`
*(runs watch but doesn't start server)*

### Starting http server
`$ gulp serve`
*(server at localhost:8080)*
